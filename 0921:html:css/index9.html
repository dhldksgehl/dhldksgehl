CSS display 속성에 대해 알아보자

  • 요소를 화면에 어떻게 보여줄건지 결정하는 속성
  • html 태그는 block과 inline 요소 나눔

inline

  1. ex: span, a, img
  2. content(내용물) 만큼 공간 할당 -> width, height, padding, margin
  3. 줄바꿈 없이 바로 옆에 배치
d-inline1 클레스를 갖는 div
d-inline2

block

  1. ex: div, p, ul, ol
  2. 한줄을 다 차지함 -> width height, padding, ,margin 사용이 자유로움
d-block 클래스를 갖는 div
d-block 클래스를 갖는 div

inline-block

  1. block요소 처럼 크기 지정을하되
  2. inline요소 처럼 다른 내용들과 같은 줄을 공유하고 싶은 경우
  3. block과 inline요소의 장점을 가짐

none

  1. 요소를 화면에 표시하지 않음
  2. 주의) display noen; 요소가 차지하는 공간도 사라짐
  3. 요소를 그리지 않음(랜더링하지않음)
